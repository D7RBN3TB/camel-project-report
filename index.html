الدوال المطلوبة:

1. Default Constructor - (مسؤولية الطالب الأول)

Camel(){ weight = 0; age = 0; name = "مجهول"; gender = 'U'; alive = false; // ميت حسب المطلوب }

هذا الكونستركتر يخلي الجمل ميت زي ما هو مطلوب في المشروع.

2. Parameterized Constructor - (مسؤولية الطالب الثاني)

Camel(float w, short a, string n, char g){ weight = w; age = a; name = n; gender = g; alive = true; // حي بالتلقائي }

هذا الكونستركتر ياخذ قيم ويخلي الجمل حي.

3. دالة Print - (مسؤولية الطالب الثالث)

void print(){ // تطبع معلومات الجمل بشكل منظم }

تطبع كل معلومات الجمل بشكل واضح.

4. دالة Age_me - (مسؤولية الطالب الرابع)

void age_me(){ if(alive == true){ age = age + 1; } else{ // رسالة خطأ } }

تزيد العمر بس لو الجمل حي.

5. دالة Eat - (مسؤولية الطالب الخامس)

void eat(){ if(alive == false){ return; // ما ياكل لو ميت } // حساب الزيادة العشوائية float percent = (rand() % 50 + 1) / 1000.0; weight = weight + (weight * percent); }

تزيد الوزن بشكل عشوائي من 0.1% إلى 5% بس لو الجمل حي.

اختبار البرنامج - (مسؤولية الطالب السادس)

اختبار 1: الكونستركتر العادي

تجربة 1: انشاء جمل بالكونستركتر العادي ======== معلومات الجمل ======== الاسم: مجهول الجنس: U العمر: 0 شهر الوزن: 0 كيلو الحالة: ميت ===============================

اختبار 2: إنشاء جمال أحياء

تجربة 2: انشاء جمال احياء ======== معلومات الجمل ======== الاسم: صحراء الجنس: F العمر: 24 شهر الوزن: 150.5 كيلو الحالة: حي ===============================

اختبار 3: دالة العمر

تجربة 3: تكبير العمر صحراء كبر شهر واحد صحراء كبر شهر واحد

اختبار 4: دالة الأكل

تجربة 4: اطعام الجمل ملك الصحراء اكل وزاد وزنه 2.5 كيلو ملك الصحراء اكل وزاد وزنه 3.1 كيلو ملك الصحراء اكل وزاد وزنه 4.2 كيلو

المشاكل والحلول

مشكلة 1: العشوائية

المشكلة: كيف نخلي الزيادة عشوائية بين 0.1% و 5%

الحل: استخدمت rand() مع srand(time(0))

مشكلة 2: منع الجمل الميت من الأكل

المشكلة: الجمل الميت ما يقدر ياكل

الحل: حطيت شرط if(alive == false) في دالة الأكل

مشكلة 3: تنظيم الإخراج

المشكلة: المعلومات تطلع مبعثرة

الحل: استخدمت خطوط الفصل والتنظيم
